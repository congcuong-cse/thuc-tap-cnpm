Nhắc đến Portable chắc mọi người hiểu rồi. Một phiên bản chạy luôn không cần cài đặt. Thậm chí với bản này các bạn có thể cài plugin bình thường (trừ 1 vài cái kiểm tra registry ngay từ đầu), còn không cứ chọn đường dẫn đến thư mục App (chứa python.exe) là được. Nếu cần nâng cấp cho lõi App này, các bạn cũng có thể cài đè bản mới vào được.

Trong này có công cụ khá hay là PyScripter (tựa tựa PyDev) dùng để soạn script hoặc thực thi các lệnh.

Một cách khác để chạy file py từ Portable Python không qua bước Open with...

Tạo file .bat có dạng như sau: 
@echo off

"D:\Portable\Python\App\python.exe" "test.py"



echo PAUSE to view results! Press any key to exit.

pause > null

Sau đó đặt chung file bat với file py. Nếu không đặt chung thì cần thêm đường dẫn vào trước file test.py.

Trong đó trên máy tớ, D:\Portable\Python\App\ là đường dẫn đến file exe chứa trong bộ Portable Python (không phải file Python-Portable.exe nhé) và test.py là tên file cần chạy.

Thêm vào menu chuột phải mục Edit with PyScripter (portable nhé):



Soạn file .reg rồi chạy, nội dung file như sau:

Nếu máy chưa cài Python: 

Windows Registry Editor Version 5.00



;Đặt setting cho .py file

[HKEY_CLASSES_ROOT\.py]

@="Python.File"

"Content Type"="text/plain"



[HKEY_CLASSES_ROOT\.py\shell]



;Chú ý sửa các đường dẫn liên quan theo máy của bạn



;Setting cho Python.File

[HKEY_CLASSES_ROOT\Python.File]

@="Python File"



[HKEY_CLASSES_ROOT\Python.File\DefaultIcon]

@="D:\\Portable\\Python\\App\\DLLs\\py.ico"



[HKEY_CLASSES_ROOT\Python.File\shell]



;Bỏ mục này nếu thấy không thích

[-HKEY_CLASSES_ROOT\Python.File\shell\Edit with IDLE]



;PyScripter chạy khá ổn :D

[HKEY_CLASSES_ROOT\Python.File\shell\Edit with PyScripter]

@="Edit with P&yScripter"



[HKEY_CLASSES_ROOT\Python.File\shell\Edit with PyScripter\command]

@="\"D:\\Portable\\Python\\App\\PyScripter.exe\" \"%1\""



;Đoạn dưới đây liên quan đến lệnh Open mặc định trong menu chính, nếu bạn không muốn mở bằng Python.exe như mặc định, hãy đổi đường dẫn, lưu ý cấu trúc và tham số của lệnh nhé.

[HKEY_CLASSES_ROOT\Python.File\shell\open]



[HKEY_CLASSES_ROOT\Python.File\shell\open\command]

@="\"D:\\Portable\\Python\\App\\python.exe\" \"%1\" %*"



[HKEY_CLASSES_ROOT\Python.File\shellex]



[HKEY_CLASSES_ROOT\Python.File\shellex\DropHandler]

@="{60254CA5-953B-11CF-8C96-00AA00B8708C}"


Nếu máy đã cài Python chỉ soạn 2 mục liên quan đến PyScripter ở trên.
